# Django RES 개발 프로젝트

## **프로젝트 시작**

### Environment
```
- python 3.9.13
- Django 4.2.8
- djangorestframework 3.14.0
```

### Install
```
$ cd Project
$ pip install -r requirements.txt
```

### Test
```
$ pip manage.py test users
$ pip manage.py test task
```
